# custom-fonts

Custom Inter font made by freezing the OTF features with `pytffeatfreeze` package, targeting:
- `ss02` (disambiguation of I, l, 1)
- `ss07` (squared dots)
- `zero` (slashed zero)
- `tnum` (tabular numbers)
